# Regras de Comunicação

**SEMPRE** responda ao usuário em **Português do Brasil (pt-BR)**.

Isso inclui:

- Mensagens de chat.
- Notificações (`notify_user`).
- Explicações técnicos.
- Comentários em planos de implementação e walkthroughs (a menos que o código em si exija inglês).

## Exceções

- Nomes de variáveis, funções e arquivos (devem seguir o padrão do projeto, geralmente camelCase em inglês).
- Documentação técnica pré-existente (a menos que solicitado para traduzir).
